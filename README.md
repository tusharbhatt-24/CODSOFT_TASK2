# Neon Tic-Tac-Toe 🎮

A beautifully designed, modern Tic-Tac-Toe game built with React Native and Expo, featuring a stunning neon UI and an AI opponent with multiple difficulty levels.

## Features ✨

- **Neon Aesthetic UI:** A visually striking interface with neon glow effects for a premium feel.
- **Player vs AI:** Play against an intelligent computer opponent.
- **Difficulty Levels:** Choose between Easy, Medium, and Hard difficulties to challenge your skills.
- **Score Tracking:** Keeps track of the scores for both you and the AI across multiple games.
- **Responsive Design:** Adapts smoothly to different screen sizes and orientations.

## Tech Stack 🛠️

- **Framework:** [React Native](https://reactnative.dev/)
- **Build Tool:** [Expo](https://expo.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## Installation & Setup 🚀

1. **Clone the repository:**
   ```bash
   git clone https://github.com/tusharbhatt-24/CODSOFT_TASK2.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd CODSOFT_TASK2
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npx expo start
   ```

5. **Run the app:**
   - Scan the QR code with the Expo Go app on your Android or iOS device.
   - Or run on an emulator/simulator by pressing `a` (for Android) or `i` (for iOS) in the terminal.

## How to Play 🕹️

1. You play as **Player X** and the AI plays as **Player O**.
2. Select your desired difficulty level (Easy, Medium, or Hard) from the bottom menu.
3. Tap on any empty cell in the 3x3 grid to make your move.
4. The first player to align 3 of their marks horizontally, vertically, or diagonally wins the game!
5. Press the "RESET GAME" button to start a new match at any time.
